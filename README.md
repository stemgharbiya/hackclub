# Gharbiya Hack Club Website

Welcome to the Gharbiya Hack Club official website! This site is designed to accept applications for our four main bootcamps.

## 🎯 Bootcamps

- **Competitive Programming** 💻 - Master algorithms, data structures, and problem-solving
- **Cyber Security** 🔒 - Learn ethical hacking and protect the digital world
- **Web Development** 🌐 - Build beautiful websites and web applications
- **Data Science** 📊 - Turn data into insights and build intelligent systems

## 🚀 Features

- Fully responsive design using Bootstrap 5
- Interactive elements powered by jQuery
- Separate application pages for each bootcamp
- Form submissions handled by FormSubmit
- Hack Club brand-compliant design
- Smooth animations and user-friendly interface
- Auto-save form data (localStorage)
- Character counters and form validation

## 🔧 Setup for GitHub Pages

1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings → Pages
4. Select the main branch as source
5. Save and wait for deployment
6. Update the `_next` URL in all form files with your actual GitHub Pages URL

## 🎯 Application Process

1. **Visit** the main page
2. **Choose** your desired bootcamp
3. **Fill out** the application form
4. **Submit** your application
5. **Receive** confirmation email
6. **Wait** for review (3-5 business days)
7. **Start learning** if accepted!
